# Resources And Copyright

Use:

- `docs/resources/resource-index.md` as the compact resource inventory.
- `docs/resources/sources-and-licenses.md` as the detailed provenance and license ledger.
- `resources/distribution/` for redistributable resources intended to ship.
- `resources/prep/` for large preparation-only resources.
- `resources/cache/` for rebuildable downloaded or generated cache.

Before using third-party code, images, audio, fonts, datasets, text, or other resources:

1. Verify the license from a primary or reliable source.
2. Confirm it allows the intended use, modification, and redistribution.
3. Record source, author, license, URL, retrieval date, intended use, and distribution status.
4. Update the compact resource index.
5. Do not use unclear, incompatible, non-commercial-only, or no-derivatives resources without explicit user approval.

Do not assume internet availability means permission.
